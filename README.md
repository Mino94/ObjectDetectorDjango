# ObjectDetectorDjango

딥러닝 활용 Minho 객체인식

*주요기능
- 실시간 영상 웹서버 전송
- 받은 영상 얼굴인식
- 인식한 객체 발견시 안드로이드 전송

*사용기술
- Framework : Django, Android studio
- Tesorflow 기반 (Darknet/Darkflow)
- Linux(Ubunt) ,VS code
- Python, html, css
- OpenCV
